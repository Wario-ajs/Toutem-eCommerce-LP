# Toutem-eCommerce-LP
Criado para o desenvolvimento do projeto Toutem (landing page de produtos naturais), para pratica de tecnologias como html/css e Javascript.

[Link do projeto no Vercel](https://toutem-e-commerce-lp.vercel.app/)
